Monitoring

  • Service status: systemctl status makerflow
  • Health endpoint: /healthz
  • Disk capacity and DB growth trend
  • Audit log and email delivery log review cadence

Security Practices

  • HTTPS only in production.
  • Strong secret key and secure cookies.
  • Role reviews for workspace admins and owners.
  • Periodic cleanup of stale accounts and sessions.

Post-Release Maintenance

  1. Run regression scripts before deploy.
  2. Run python3 scripts/sync_website_content.py.
  3. Review release notes page for external communication.
