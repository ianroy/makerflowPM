Getting Started

Repository: github.com/ianroy/makerflowPM

Startup

git clone https://github.com/ianroy/makerflowPM.git
cd makerflowPM
cp .env.example .env
python3 app/server.py

Open http://127.0.0.1:8080/login.

Default Account (Rotate Immediately)

  • admin@makerflow.local / ChangeMeMeow!2026

Core User Roles

  • Owner: global governance and high-risk admin actions.
  • Workspace Admin: manages one workspace/department.
  • Manager/Staff/Student/Viewer: scoped by permissions and nav visibility.

First-Day Setup Checklist

  1. Create workspaces for each department or service unit.
  2. Create teams and spaces (labs, studios, branches, rooms).
  3. Define default statuses and labels by board.
  4. Import current project/task data from CSV.
  5. Set user roles and email preference defaults.
